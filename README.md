# HorizonCrypt
Animal Crossing: New Horizons Save Encryptor/Decryptor

## Usage
HorizonCrypt \[-b\] \[-c|-d\] \<input\>

## Examples

#### Decrypt File
HorizonCrypt -d ./main.dat

#### Encrypt File
HorizonCrypt -c ./main_decrypted.dat

#### Decrypt Folder
HorizonCrypt -b -d ./

#### Encrypt Folder
HorizonCrypt -b -c ./
